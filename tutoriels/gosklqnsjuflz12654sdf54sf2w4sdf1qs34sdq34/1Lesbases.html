1 - Les bases

LA PROGRAMMATION BASH



LES BASES


Dans ce chapitre, nous allons voir les bases de la programmation bash qui permet de créer des automatisation de tâches dans Linux.


Pour commencer exécuter les 3 commandes suivante pour installer ou mettre a jour l'interpréteur bash:

  • apt-get update
  • apt-get upgarde
  • apt-get install bash


Pour commencer, le fichier doit comporter obligatoirement cette ligne de code en début de fichier pour indiquer le type de langage de programmation:


#!/bin/bash


#Code du programme

PS: Les couleurs du programme sont identique à celle de Notepad++.


A noter que la seconde ligne est un commentaire, elle permet de commenter son code et ces lignes sont ignorer par l'interpréteur lors de lancement du fichiers.
