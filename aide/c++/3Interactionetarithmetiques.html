Les constantes :


Les constantes sont des variables qui une fois déclarer ne peuvent pas changer de valeurs. Cela peut servir pour conserver des nombres qui ne changeront pas comme ∏.


Exemple:

  • double const pi(3.14159);
  • double const g(9.81);
