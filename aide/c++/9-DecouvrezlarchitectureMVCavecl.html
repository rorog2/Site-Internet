9 - Découvrez l'architecture MVC avec les widgets complexes






L'architecture simplifié modèle/vue de Qt :


Les classes gérant le modèle :


  • QStringListModel : une liste de chaîne de caractères, de type QString. Très simple, très basique;
  • QStandardItemModel : une liste d'éléments organisé sous forme d'arbre (chaque élément peut contenir des sous-éléments);
  • QDirModel : la liste des fichiers et dossiers stockés sur votre ordinateur. Ce modèle va, en arrière plan, analyser votre ordinateur pour être prêt à l'emploi;
  • QSqlQueryModel, QSqlTableModel, QSqlRelationalTableModel : donnée issu d'une base de donnée.



Les classes gérant la vue :


  • QListView : une liste d'éléments

  • QTreeView : un arbre d'éléments, où chaque éléments peut avoir des éléments enfants

  • QTableView : un tableau
